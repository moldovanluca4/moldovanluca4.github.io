04. Featured Projects

Hackathon Winning Project - Market Research Automation Agent

Built an award-winning AI automation agent for market research that discovers EdTech/LMS competitors, extracts company websites from blogs and search results, and enriches the dataset with structured LinkedIn information such as company size, headquarters, industry, products, and specialties. The project also collects forum discussion links from platforms like Reddit and Quora to analyze authentic user opinions, complaints, pros, and cons. The final data is cleaned, consolidated into a SQLite master table, and prepared for integration with n8n workflows and Google Sheets.

Python Playwright Pandas SQLite

Internship Project - Mainframe

Built a Flask-based web application that integrates a C program into a Python workflow, processes the program’s output, stores the results in an SQLite database, and displays them through a web interface. The project was designed as a self-contained system where Flask acts as the middle layer between the C executable, the database, and the frontend, making the application feel like one cohesive software product rather than separate scripts. Through this project, I explored Flask, SQLite, SQL, Python-C integration, Linux runtime environments, and web interface design.

IBM Z/os C Programming Language Python Flask Linux

University Databases Project

Built the Campus Information Center (CIC) as part of the Fall 2025 Database course, a database-driven Flask web application for managing and visualizing a university campus ecosystem. The project focused on clean relational database design, using normalized SQL schemas and ISA hierarchies to model buildings, events, academic departments, lecture halls, research facilities, and campus personnel. The application combines a Flask backend, SQL database, and HTML/CSS/Jinja2 frontend to provide a structured platform for managing campus resources and academic information.

Python HTML SQL Flask
